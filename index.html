Font sizes

How to use

<p class="font-x">Text</p>

x: can be any number b/w 8-50px

<p class="font-14">Text</p>

Add a number between the above range and get desired font-size for text elements.

Placeholder

How to use

.input-field{
    @include placeholder{
        color:red;
        font-size:12px;
    }
}

Suppose we have to change the styles of placeholder in our input field, just include Placeholder mixin in the input field (selected by class/id) and change the properties of placeholder accordingly.

Font Family

How to use

@include font-face("Open Sans", "../fonts/OpenSans-Light", 300);
@include font-face("Open Sans", "../fonts/OpenSans-Regular", 400);
@include font-face("Open Sans", "../fonts/OpenSans-SemiBold", 600);
@include font-face("Open Sans", "../fonts/OpenSans-Bold", 700);
@include font-face("Open Sans", "../fonts/OpenSans-ExtraBold", 800);

Mixin accepts 3 parameters.

  • Name
  • Path
  • Weight
