Font sizes

How to use

// base/_variable.scss

$fontSizeRange:( 
    min:8, //default min value (8px).
    max:50 //default max value (50px).
);
:: Set you min / max font range in "base/_variable.scss". 
<p class="font-x">Text</p>
x: can be any number b/w your min / max range defined 
<p class="font-14">Text</p>

Add a number between the above range and get desired font-size for text elements.

Placeholder

How to use

.input-field{
    @include placeholder{
        color:#1478FF;
        font-size:12px;
    }
}

Just include Placeholder mixin in the input selector (selected by class/id) and change the properties of placeholder accordingly.

Font Family

How to use

//Light 
@include font-face("Open Sans", "../fonts/OpenSans-Light", 300);

//Regular
@include font-face("Open Sans", "../fonts/OpenSans-Regular", 400);

//Semibold
@include font-face("Open Sans", "../fonts/OpenSans-SemiBold", 600);

//Bold
@include font-face("Open Sans", "../fonts/OpenSans-Bold", 700);

//Extra Bold
@include font-face("Open Sans", "../fonts/OpenSans-ExtraBold", 800);

Just Copy the above code and add where you want to define a particular font.


Mixin accepts 3 parameters.

  • Font Name (e.g "open sans")
  • Font Path (e.g "../fonts/OpenSans-Bold")
  • Font Weight (e.g "400")

Buttons







How to use


In HTML: 
<button class="success-btn large">large btn</button> 

<button class="success-btn">Regular btn</button> 

<button class="secondary-btn">Secondary btn</button> 
 
<button class="cancel-btn">Cancel Btn</button> 

<button class="success-btn" disabled>Disable Btn</button>

In SCSS:

Just Copy the above code and add where you want to style a particular button.



Note:: Values shoud be added in same sequence as parameter in mixin. Sequence is mentioned below 

@include btn(padding, background-color, font-size, border-radius, text-color, border-color)

.success-btn{
    @include btn(12px 20px, map-get($color, primary), 16px, 5px, #fff, transparent);
   
    &:hover{
        background: map-get($color, primary-hover);
    }
}


Disable button parameter sequence

@include disable(background-color, text-color, border-color);

Media Queries

How to use - 

For mobile phone use for-phone-only
For table/Ipad use  for-tablet
For Desktop use  for-desktop
example given below -:


.topic-title{
    font-size: 24px;
    @include for-phone-only {    
        font-size: 20px;
    }
    @include for-tablet {    
        font-size: 22px;
    }
    @include for-large-desktop{
        font-size: 28px;
    }
}

Borders

How to use

HTML

<div class="custom-box"></div>

                    
SCSS
.custom-box{
    width: 150px;
    height: 150px;
    @include border(1px,solid,#cdcdcd);
}

To add direction
@include border(1px,solid,#cdcdcd,top/left/right/bottom);


NOTE:By default direction parameter has All value, i.e Borders will applied on all 4 sides.
Mixin accepts 4 parameters.
  • Width
  • Border style
  • Color
  • Direction(optional)

Working Example


@include border(1px,solid,black);

@include border(1px,solid,black,top);

@include border(1px,solid,black,left);

@include border(1px,solid,black,right);

@include border(1px,solid,black,bottom);
