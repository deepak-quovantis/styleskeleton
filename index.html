Font sizes

How to use

<p class="font-x">Text</p>

x: can be any number b/w 8-50px

<p class="font-14">Text</p>

Add a number between the above range and get desired font-size for text elements.

Placeholder

How to use

.input-field,
    textarea{
    @include placeholder{
    color:red;
    font-size:12px;
    }
}

Suppose we have to change the styles of placeholder in our input field, just include Placeholder mixin in the input field and change the properties of placeholder accordingly.
